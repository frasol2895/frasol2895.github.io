# SDC260 Responsive Website Project

## Description
This is my responsive website project for SDC260 built with HTML5 and CSS3. The website features a clean, modern design that works well on desktops, tablets, and mobile devices.

## Features
- Fully responsive layout using CSS media queries

## How to Run Locally with XAMPP
1. Install XAMPP on your computer.
2. Copy the project files (index.html and styles.css) into the `htdocs` folder.
3. Start the Apache server in XAMPP.
4. Open your browser and go to `http://localhost/your-folder-name` (or `http://localhost` if files are placed directly in htdocs).
5. The website should load locally.
